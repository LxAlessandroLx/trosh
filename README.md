# Trosh
A fork of [trosh](https://github.com/Stabyourself/trosh) with some improvements such as support for LÖVE 11

## Required libraries
[LÖVE](https://love2d.org/) 11.0 - 11.4

## Installation
<pre>
$ git clone https://github.com/LxAlessandroLx/trosh.git
$ cd trosh
$ make
</pre>
